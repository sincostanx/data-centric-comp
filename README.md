# Data-Centric AI Competition
This repository provides altered roman numeric dataset and code used during the [Data-Centric AI Competition](https://https-deeplearning-ai.github.io/data-centric-comp/) in Python. I improved model accuracy from the baseline of 0.64421 to 0.81240, placing at ranking 103/516.

* Manually correcting mislabeled and removing inappropriate samples from the original data
